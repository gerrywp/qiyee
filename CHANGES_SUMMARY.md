## 📝 修改的文件列表

### 后端文件修改

#### 1. `api/models/banner.go`
**修改内容**: 添加缩略图字段
```go
type Banner struct {
    gorm.Model
    Url      string // 原始图片URL
    ThumbUrl string // 缩略图URL (新增)
}
```

#### 2. `api/models/initDB.go`
**修改内容**: 添加自动迁移
- 添加 `AutoMigrate()` 调用以自动创建/更新表结构
- 支持 Banner, User, About, Prod 等所有模型

#### 3. `api/service/upload.go`
**修改内容**: 核心功能增强
- 新增 `BannerUploadResult` 结构体
- 新增 `UploadWithThumbnail()` 方法 - 上传并生成缩略图
- 新增 `GenerateThumbnail()` 方法 - 生成指定尺寸的缩略图
- 新增 `CropAndSaveImage()` 方法 - 处理裁切图片（预留）
- 新增 `ResizeImage()` 方法 - 图片缩放（预留）
- 依赖: `github.com/disintegration/imaging`

#### 4. `api/service/banner.go`
**修改内容**: 使用新的上传方法
- 修改 `Upload()` 方法以使用 `UploadWithThumbnail()`
- 新增 `CropImage()` 方法处理裁切（预留）
- 同时保存原图和缩略图 URL

#### 5. `api/router/routers.go`
**修改内容**: 路由和处理函数更新
- 修改 `banner()` 函数 - 传递完整 Banner 对象
- 修改 `bannerUpload()` 函数 - 返回 JSON 响应
- 新增 `bannerCrop()` 函数 - 处理裁切（预留）
- 新增 `/pai/banner/crop` 路由

### 前端文件修改

#### 6. `web/views/backend/layouts/header.tmpl`
**修改内容**: 添加 Cropper.js CSS
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/cropperjs/1.5.13/cropper.min.css">
```

#### 7. `web/views/backend/layouts/footer.tmpl`
**修改内容**: 添加 Cropper.js JavaScript
```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/cropperjs/1.5.13/cropper.min.js"></script>
```

#### 8. `web/views/backend/includes/banner.tmpl`
**修改内容**: 完全重写
- 新的 3 列响应式布局
- 每列使用独立的卡片组件
- 集成 Cropper.js 裁切功能
- 两个 Modal: 裁切 Modal 和预览 Modal
- 完整的 JavaScript 交互逻辑
  - 文件上传
  - 图片预加载
  - Cropper.js 初始化
  - 裁切和上传
  - 数据回显
  - 错误处理

### 配置文件修改

#### 9. `go.mod`
**修改内容**: 添加依赖
- `github.com/disintegration/imaging v1.6.2`
- `golang.org/x/image v0.0.0-20191009234506-e7c1f5e7dbb8`

## 🔄 工作流程

### 用户交互流程
```
┌─────────────────────────────────────────────────┐
│                 用户操作                          │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│         选择"上传图片"按钮                        │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│    弹出文件选择器，选择图片文件                   │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│    图片加载到 Cropper Modal（带预览）            │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│    用户在 Cropper 中调整裁切区域                 │
│    - 拖动移动图片                               │
│    - 滚动缩放                                    │
│    - 调整裁切框大小                              │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│      点击"确认裁切"按钮                          │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│   前端: 使用 Canvas 导出裁切区域                 │
│   生成 Blob 数据                                │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│   前端: AJAX 上传到后端 /pai/banner/upload       │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│  后端: 接收文件并保存为原始图片                   │
│   FileURL: /static/upload/1234567890.jpg        │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│  后端: 自动生成缩略图 (400x300)                   │
│   ThumbURL: /static/upload/1234567890_thumb.jpg │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│    后端: 保存两个 URL 到数据库                    │
│    UPDATE banners SET url=?, thumb_url=?        │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│   后端: 返回 JSON 响应                           │
│   {code: 1, msg: "上传成功"}                    │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│   前端: 显示成功提示，关闭 Modal                 │
│   使用 Canvas 数据更新页面预览                   │
└─────────┬───────────────────────────────────────┘
          │
          ▼
┌─────────────────────────────────────────────────┐
│   1秒后自动刷新页面，显示新的缩略图              │
└─────────────────────────────────────────────────┘
```

## 🔄 部署注意事项

1. **安装依赖**
   ```bash
   go get github.com/disintegration/imaging
   ```

2. **数据库迁移**
   - 首次启动时会自动创建/更新表结构
   - 旧数据库会自动添加新的 `thumb_url` 字段

3. **文件权限**
   - 确保 `web/static/upload` 目录存在且有写入权限

4. **编译和运行**
   ```bash
   go build -o qiyee ./cmd/main.go
   ./qiyee
   ```
