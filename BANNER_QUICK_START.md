## 📸 功能使用

### 上传图片
1. 在 Banner 管理页面，点击任一 Banner 卡片的 **"上传图片"** 按钮
2. 在弹出的文件选择器中选择要上传的图片（支持 jpg, png, gif, jpeg, bmp）
3. 图片会自动加载到裁切框

### 裁切图片
1. 图片加载到裁切框后，你可以：
   - **拖动** 图片调整位置
   - **滚动** 鼠标缩放图片
   - **调整裁切框** 的大小和位置
   - 使用 **网格和辅助线** 获得精确的构图

2. 点击 **"确认裁切"** 按钮保存裁切结果

### 查看原图
1. 点击任一 Banner 的 **"预览原图"** 按钮
2. 弹出 Modal 显示原始大图

### 美观的布局
- **3列设计**: 三个 Banner 平均分布在屏幕宽度

- 移动设备上会自动调整布局

## 💡 常见问题

### Q: 上传失败，显示"参数错误"
**A:** 请确保：
- 已选择一个有效的图片文件
- 文件格式为 jpg, png, gif, jpeg 或 bmp
- 文件大小不超过 5MB

### Q: 裁切框没有显示
**A:** 请检查：
- 浏览器是否支持 JavaScript（应该启用）
- Cropper.js 库是否成功加载（查看浏览器控制台）
- 网络连接是否正常（CDN 可访问）

### Q: 上传后缩略图显示不正确
**A:** 可能的原因：
- 缩略图文件未生成（检查 imaging 库是否正确安装）
- 文件权限问题（检查 `web/static/upload` 目录权限）
- 数据库未更新（重启服务器）

### Q: 如何修改缩略图尺寸？
**A:** 编辑 `api/service/upload.go`，找到以下行：
```go
thumb := imaging.Fill(src, 400, 300, imaging.Center, imaging.Lanczos)
```
将 `400` 和 `300` 改为你想要的宽度和高度。

### Q: 如何删除某个 Banner？
**A:** 当前版本不支持删除功能，你可以上传一张新图片来覆盖旧图片。删除功能可在后续版本中添加。

## 📝 API 文档

### 上传 Banner 图片
```
POST /pai/banner/upload

参数:
  - id: Banner ID (1-3)
  - file: 图片文件 (multipart/form-data)

响应:
  {
    "code": 1,
    "msg": "上传成功"
  }
```

### 获取 Banner 数据
```
GET /pai/banner

响应 HTML 页面，包含 Banner 管理界面
```

## 🐛 调试技巧

### 启用详细日志
在服务器启动时，你会看到详细的 GORM 日志和 Gin 路由信息

### 检查缩略图是否生成
1. 打开浏览器开发工具 (F12)
2. 在 Network 标签中查看图片请求
3. 缩略图 URL 应该类似于: `/static/upload/1234567890_thumb.jpg`

### 检查数据库
由于使用 SQLite，你可以使用 SQLite 浏览器查看：
- 打开 `pai.db3` 文件
- 查看 `banners` 表
- 确认 `url` 和 `thumb_url` 字段都已保存
